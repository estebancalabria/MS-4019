# MS-4019 - Transformación de los Procesos Empresariales Cotidianos con Agentes

# 📘 Introducción

Microsoft 365 incorpora agentes impulsados por Inteligencia Artificial para transformar la manera en que las organizaciones trabajan, colaboran y administran la información.

Estos agentes se integran principalmente en:

* Microsoft 365 Copilot Chat
* SharePoint
* Microsoft Teams
* OneDrive
* Otras herramientas del ecosistema Microsoft 365

Los agentes utilizan:

* Inteligencia Artificial
* Machine Learning
* IA Generativa
* Microsoft Graph

para comprender contexto, analizar información y automatizar tareas.

---

# 🤖 ¿Qué Son los Agentes?

Los agentes son herramientas inteligentes diseñadas para asistir a los usuarios y automatizar procesos empresariales.

## Capacidades

* Automatizar tareas rutinarias.
* Responder consultas.
* Generar contenido.
* Analizar información.
* Organizar datos.
* Ayudar en procesos empresariales.

A diferencia de un asistente tradicional, los agentes pueden:

* Trabajar junto al usuario.
* Actuar en nombre del usuario.
* Ejecutar tareas complejas.
* Automatizar procesos de múltiples pasos.

# 🌐 Datos Web vs Datos de Trabajo

## Datos Web

Información pública accesible desde Internet.

## Datos de Trabajo

Información empresarial almacenada en:

* SharePoint
* Microsoft Graph
* Conectores Graph

⚠️ No incluyen correos personales ni chats privados.
